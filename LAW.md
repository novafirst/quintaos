# QuintaOS Kernel Law v0.1

This document defines the immutable laws of existence inside QuintaOS.

## Status
Frozen. Immutable. Historical.

## Guarantees
- Capability-based authority only
- No ambient trust
- No implicit privilege
- Message-only interaction

## Non-Goals
- POSIX compatibility
- Shells or users
- Filesystems
- Human-first abstractions

## Warning
This project is not an operating system in the traditional sense.
It is a substrate for autonomous computation.
